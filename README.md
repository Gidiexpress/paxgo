# PaxGo

Welcome to the **PaxGo** project. This is an AI-powered mobile application built with React Native and Expo.

## Getting Started

1. **Install dependencies:**

    ```bash
    npm install
    ```

2. **Start the app:**

    ```bash
     npx expo start
    ```

## Architecture

* **Frontend:** React Native (Expo Router)
* **Backend / Auth:** Supabase
* **AI:** Groq SDK
* **Payments:** RevenueCat
